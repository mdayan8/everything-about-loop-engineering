# Module 09: Advanced Topics

> **Multi-loop coordination, token economics, and applying the loop pattern beyond coding.**

---

## What's in This Module

Once you have a few loops running, new challenges emerge:

- **Multi-loop coordination** — what happens when two loops touch the same repo?
- **Token economics** — why costs spike and how to budget
- **Beyond coding** — the generate→verify→refine pattern applied outside software

These topics are covered in separate files:

1. [Multi-Loop Coordination](multi-loop-coordination.md)
2. [Token Economics](token-economics.md)
3. [Beyond Coding](beyond-coding.md)

---

## When to Read These

- **Multi-loop coordination**: when you have 2+ loops running on the same repo
- **Token economics**: when your loop costs are higher than expected, or you're planning a high-frequency loop
- **Beyond coding**: when you want to apply the loop pattern to non-software tasks
